# Inventory Object Detection and Product Counting using YOLOv5 and SKU110K

## Overview

This project implements an automated inventory object detection and product counting system using YOLOv5 and the SKU110K retail dataset.

## Dataset

SKU110K Dataset

* Training Images: 8233
* Validation Images: 588
* Test Images: 2941

## Technologies Used

* Python
* PyTorch
* YOLOv5
* OpenCV
* Pandas
* NumPy
* Matplotlib

## Features

* Product Detection
* Inventory Counting
* Bounding Box Visualization
* CSV Report Generation
* Statistical Analysis
